# papildoma-uzduotis

## Įdiegimo instrukcija

### Sukompiliuotos programos paleidimas
1. Atsisiųskite programos *.exe* failą paspaudę [čia](https://github.com/eimantaskat/papildoma-uzduotis/releases/download/v1.0/main.exe)
2. Palieskite atsisiųstą failą

### Programos sukompiliavimas
1. Atsisiųskite programos kodą paspaudę [čia](https://github.com/eimantaskat/papildoma-uzduotis/archive/refs/tags/v1.0.zip) arba su komanda
```console
$ git clone https://github.com/eimantaskat/papildoma-uzduotis.git
```
2. Sukompiliuokite programą
```console
$ make
```
arba  
```
$ g++ -o <pavadinimas> main.cpp container_functions/*.cpp functions/*.cpp student/*.cpp person/*.cpp -O3
```
3. Paleiskite programos *<pavadinimas>.exe* failą

## Naudojimosi instrukcija
1. Palieskite programos ***.exe** failą
2. Įveskite failo, kuriame yra tekstas, pavadinimą
3. Pasirinkite ką norite daryti - išvesti rezultatą į failą, sugeneruoti lentelę ar abu
4. Rezultatai bus išvesti į faius

## Programos failų struktūra
* **.cpp**:
  * main.cpp - pagrindinis programos failas, iš kurio kviečiamos kitos programos funkcijos
  * functions.cpp - pagrindinių funkcijų failas (failo skaitymo, pasikartojančių žodžių radimo ir išvedimo, lentelės generavimo funkcijos)
  * helper_functions.cpp - pagalbinės funkcijos
* **.hpp**:
  * main.hpp - visi *#include* ir *using*
  * functions.hpp - funkcijų aprašymai
